# anthroposophy-events-calendar
A calendar of Anthroposophical events in and around Edinburgh.

## Traffic tracking

This site now supports Cloudflare Web Analytics traffic tracking via `analytics.js`.

1. In Cloudflare, open Web Analytics and create/copy your site token.
2. Open `analytics.js` and set `token`.
3. Deploy the site changes.

When `token` is set, Cloudflare will track website traffic and page views.
